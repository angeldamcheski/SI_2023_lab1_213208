1. Прво се прави фолдер во кој што ќе работиме.
2. Го иницијализираме git во фолдерот со командата git init
3. Додаваме README.md датотека во нашиот фолдер и јава класа
4. Со командата git branch правиме три гранки bug1, bug2, bug3 
5. Со git checkout се префрламе на сите гранки и ги поправаме грешките во ПДФ документацијата соодветно
6. Правиме push на remote репото и ги синхронизираме сите гранки
7. Со git merge ги спојуваме bug1 и bug2 со гранката master
8. Со git rebase ја преместуваме master на bug3.
9. Како последен чекор правиме push на сите досегашни датотеки со commit порака "Final commit".
